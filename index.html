FIMM GOOGLE TRENDS PROJECT

Info

This website is a tool to illustrate the seasonality of google searches of mood, interest of doing exercise and observe the weather conditions in those seasonal changes.

User Manual

Choosing categories and regions.

You can compare differences between countires by selecting them from the buttons below the respective countrys flag.
Then choose Data categories you want to see on the chart with the buttons below their names.
To remove Data categories or a country from the chart simply uncheck the selected country or category you want gone.

Categories are marked with titles.
"TRENDS" is for Google Trends data.
"WEATHER"" is for MetOffice weather data.

Start date and End date.

The default time scope to compare in the chart is January 2006 as the start date and December 2019 as the end date.
To change start date select the month by clicking the arrow facing down of the current selected start month button and click the month you want from the list that opens.
Then change start year by clicking the the downward facing arrow of the current selected start year button and click the year you want from the list that opens.
Selecting the end date works the same way as selecting the start date.
